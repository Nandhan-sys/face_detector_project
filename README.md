Driver Attention Detection

This project detects driver attention levels using machine learning. The model predicts whether a driver is **focused** or **distracted** based on real-time face detection data.

### Files:

1. **_collect_data.py**: Collects data by detecting the face from a webcam feed.
2. **train_model.py**: Trains a decision tree model to classify face data as focused or distracted.
3. **live_prediction.py**: Runs the model on a live webcam feed to predict driver attention.

### Technologies Used:
- OpenCV
- Scikit-learn
- Python
